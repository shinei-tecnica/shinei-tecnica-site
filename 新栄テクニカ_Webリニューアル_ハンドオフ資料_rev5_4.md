## 1. 背景と最終ゴール

- 本業はプラスチック射出成形用金型・ダイカスト金型の設計/製作（名古屋市天白区）。
- 課題: 構造的な売上減・紹介依存・機械稼働のムラ。
- **最終ゴール: 現場稼働率・新型獲得・機械稼働率を上げ、高収益体制へ。**
- 入口として **Web と SNS を最大活用**する。

## 2. 全体戦略（2エンジン × 1サイト）

```
■ 売上エンジン ── 金型課題解決（入口①・主役）
    原因特定 → 修理 → 設変 → 型の作り替え → 新型
■ 成長エンジン ── ENTOEN（入口②・副）
    循環 ／ 製品化 ／ スポンサー ／ 実証
```

- **売上エンジン**: 「金型トラブルの原因特定から修理・設変まで対応します」を起点に5段ラダーで引き上げる。
- **成長エンジン（ENTOEN）**: 金型を知らない層を取り込む将来開拓。
- 1ドメインに同居。**売上エンジンの5段は"はしご"＝修理→新型の転換動線そのもの**。

## 3. ロードマップ

```
Phase 1（今）     金型 80% ／ ENTOEN 20%
Phase 2（POC後）  昭和学院 成功 → スポンサー獲得 → ENTOEN 拡大
```

## 4. ポジショニング

アンブレラ: プラスチック製品の製造技術を活かし、**金型の課題解決**と**プラスチック循環の実装**を行う。

入口①（金型）3本柱:
1. 成形不具合の**原因特定**（診断・技術力）＝主役
2. **二重の近さ** ＝ 名古屋拠点＋汎用部品の即日手配体制
3. プラ射出の**難物特化** ＝ 他社製・現品（図面なし）

入口②（ENTOEN）: 金型用語ゼロ・成果ベース・物語性。※ダイカストは新型のみ。

## 5. 競合差別化と禁止事項（厳守）

恵那三洋製作所（愛知も商圏の競合）との表現重複を厳禁。
- 避ける名/メタファー/言い回し: 金型パック／金型119番／金型パドック、消防・病院・競走馬系、「図面レス」「丸ごとおまかせ」「ライン止めない」「預かって成形」、数値模倣、愛称/キャラ系
- 視覚: 先方はカラフル → 本サイトは抑制的・インダストリアル

## 6. サイト構成（1ドメイン・2入口）

### トップ（金型主役の振り分け）
- メインコピー: **「金型トラブルの原因特定から修理・設変まで対応します」**
- 主導線「金型でお困りの方 ▶」／副導線（控えめ）「廃材・循環から製品をつくりたい方 ▶ ENTOEN」
- **「こんな方に向いています」セクション**（「自分が相談してよい会社か」を即判断させる）:
  - □ 金型課題: 他社製金型でお困りの方／図面が残っていない金型をお持ちの方／成形不良の原因が分からない方／修理・設変・新型を相談したい方
  - □ 循環・製品化: 自社の廃材活用方法を検討している方／廃材を活用したオリジナル製品を作りたい方／ESG・CSR活動を形にしたい方
- **「ご相談から対応までの流れ」セクション**（後述⑧）

### 入口① 金型・修理ハブ（売上エンジン／5段ラダー）

- 予防保全（定期点検）＝継続収益
- **「対応が難しいケース」セクション**（後述⑦。金型相談フォーム直前に配置）

- 新規金型製作（新型。プラ射出＋ダイカスト〈対応可能な範囲〉。ラダー到達点）

### ⑥ 金型相談フォーム（旧「お問い合わせ」を構造化フォームに変更）
- ページ名: **「金型相談フォーム」**（または「修理・設変・新型 相談フォーム」）
- 目的: 一般営業メールを減らし、相談案件の比率を高める。
- 入力項目:
  - ご相談内容（複数選択可）: 修理 / 設変 / 新型 / 成形不良の原因調査 / その他
  - 現在の症状（複数選択可）: バリ / ショート / 反り / 離型不良 / ピン折れ / その他
  - 他社製金型か（はい / いいえ / 不明）
  - 図面の有無（あり / なし / 不明）
  - 希望時期（テキストまたは選択）
  - 写真添付（ファイルアップロード）※技術的に要件あり、後述
  - お名前 / 会社名 / メールアドレス / 電話番号
- LINE＋電話タップも同ページに併記（フォームに加えて即レス手段）
- ※ 写真添付のファイルアップロードは Netlify Forms が標準対応（Cloudflare Pagesは非対応）。**この要件によりホスティングは Netlify を推奨**。

### ⑦ 「対応が難しいケース」（金型ハブ内、金型相談フォーム直前に配置）
- セクション名: **対応が難しいケース**
- 内容:
  - 成形品・金型の現物確認ができない案件
  - 要求仕様や精度条件が不明確な状態での保証依頼
  - 品質確認期間を確保できない極端な短納期案件
  - 法令・契約上の理由で詳細確認ができない案件
- 補足文: 「まずは状況を確認させていただき、対応可否を判断いたします。」
- 目的: 全部受ける会社ではなく、**判断基準を持った会社**として見せる。

### ⑧ 「ご相談から対応までの流れ」（トップ・金型ハブ・金型相談フォームに配置）
- セクション名: **ご相談から対応までの流れ**
- 内容:
  1. **写真・内容確認** … まずは不具合箇所や状況を確認します。
  2. **概算可否判断** … 写真だけで判断できる範囲をお伝えします。※写真のみでは確定見積ができない場合があります。
  3. **現物確認** … 必要に応じて金型や成形品を確認します。
  4. **お見積** … 対応内容をご提案します。
  5. **修理・設変・新型対応** … 内容に応じて対応を進めます。
- 目的: 問い合わせ前の心理的ハードルを下げる。初めての相談でも何が起きるか見通せる。
